# Battleship

A classic Battleship game built with HTML, CSS, and JavaScript. Play against the computer on a 10×10 grid.

## Features

- Two boards: player (ships visible) and enemy (ships hidden)
- Ships of sizes 5, 4, 3, 3, 2 placed randomly
- Click enemy cells to attack; X marks hits, gray marks misses
- Tracks wins, losses, and total moves
- Score persists across sessions via localStorage

## How to Play

1. Open `index.html` in a browser
2. Click **Start Game**
3. Click cells on the enemy board to fire
4. Sink all 5 enemy ships to win

## Files

- `index.html` — Game layout
- `style.css` — Styling
- `game.js` — Game logic and AI
